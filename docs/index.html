pyControl D-Series Breakout¶

pyControl D-Series Breakout clipped onto DIN rail

Top view

What is this?¶

pyControl D-Series breakout is a PCB that connects a pyboard D-series microcontoller with twelve RJ45 ports. It is intended to work with the pyControl behavior control framework by using a microcontroller running Micropython and having the RJ45 connections that are pin compatible with pyControl Devices.

How is it different?¶

The core distinction between this breakout board and the official pyControl breakout board 1.2 is the use of a newer generation pyboard. Aside from being smaller, faster, and having more memory, the major benefit of using the D-Series pyboard is having access to more I/O pins. Having more pins enables the breakout to provide more behavior ports on a single smaller PCB without the need to add a port expander.
